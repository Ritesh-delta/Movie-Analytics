# 🎬 Movie Analytics Project

> 📊 An Exploratory Data Analysis (EDA) project on a movie dataset using **Python**, **Pandas**, **Matplotlib**, **Seaborn**, and **Power BI** to uncover insights about movie popularity, ratings, genres, languages, and release trends.

---

## 🚀 Project Overview

This project focuses on analyzing a movie dataset to discover meaningful insights through data cleaning, preprocessing, visualization, and interactive dashboards.

The analysis answers important business questions such as:

- 🎭 Which genre appears most frequently?
- ⭐ Which vote category contains the most movies?
- 🔥 Which movie has the highest popularity?
- 📉 Which movie has the lowest popularity?
- 🎬 Which movies have the highest average ratings?
- 📅 Which release year produced the most movies?
- 🌍 What are the most common original languages?

## 💡 Key Insights

- 🎭 **Drama** is the most frequent genre in the dataset.
- ⭐ The **Popular** vote category contains the highest number of movies.
- 🔥 **Spider-Man: No Way Home** is the most popular movie.
- 📉 The movie with the lowest popularity was identified through the analysis.
- 📅 **2020** recorded the highest number of movie releases.
- 🌍 **English** is the dominant original language, followed by Japanese and Spanish.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Ritesh-delta/Movie-Analytics.git
```

2. Navigate to the project directory

```bash
cd Movie-Analytics
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

4. Open the Jupyter Notebook

```bash
jupyter notebook
```
